Html - Elementy jakie używam

  • <html> = <head>+<body>
  • <main> - Główna część dokumentu
  • <section> - Sekcja dokumentu
  • <article> - Samodzielne treści
  • <header> - Nagłówek - dokumentu / sekcji / artykułu
  • <footer> - Stopka - dokumentu / sekcji
  • <h1-h6> - Tekstowe nagłówki tytułowe - dokumentu /sekcji
  • <p> - Paragraf / akapit
  • <br> - nowa linia
  • <b>/pogrubienie oraz <i>/wykrzywiony tekst - Nie mają dużego znaczenia semantycznego - służą do wyróżnienia fragmentu z tekstu otaczającego
  • <strong> - podkreśla wysoką istotność i poważność frazy w nim zawartej
  • <em> - podkreśla, że tekst może mieć inne znaczenie (np. ironia)
  • <a> - linki / łącza wewnątrz dokumentu
  • <element id="moj-identyfikator"> - atrybut id - identyfikuje element (przydaje się do nawigacji w obrębie dokumentu)
  • <nav> - Nawigacja
  • <img src="" alt=""> - img - obrazek o podanym adresie
  • <sup> - index górny
  • <sub> - index dolny
  • <ul> - lista nieuporządkowana
  • <li> - element listy
  • <ol> - lista uporządkowana
  • <dl> - Description List:
  • <dt> - Description term
  • <dd> - Description detail

BEM

Block Element Modifier

Co oznacza BEM:

  • Block - nadrzędny element
  • Element - część bloku (nie musi być bezpośrednim dzieckiem)
  • Modifier - modyfikator - jeśli element występuje w kilku wariacjach tworzymy dodatkową klasę

Markdown

Język znaczników przeznaczony do formatowania tekstu zaprojektowany przez Johna Grubera i Aarona Swartza. Został stworzony w celu jak najbardziej uproszczonego tworzenia i formatowania tekstu.

Znaczniki

  • # - nagłówek (odpowiednik h1)
  • ## - nagłówek (odpowiednik h2) i tak dalej i tak dalej
  • [opis linku](link) - odpowiednik html-owego a(link)
  • ![odpowiednik treści alt=""](link do obrazu) - odpowiednik obrazu img
  • Kwadratowe [] przed okrągłymi () - K przed O (alfabetycznie) - podstawowa reguła
  • `polecenie / jednolinijkowy kod` - wyróżniona treść
  • ```javascript blok kodu ``` - tekst w bloku. W zależności od przeglądarki lub programu wpisując na początku kod (np. javascript) dostajemy kolory z edytora
  • *tekst* - kursywa
  • **tekst** - pogrubienie
  • ***tekst*** - kursywa i pogrubienie
  • > tekst - wcięcie do cytatu
  • - tekst - lista
  • 1. tekst ...i tak dalej - lista numeryczna
  • .md - pliki Markdown mają rozszerzenie .md

Bootstrap

Bootstrap to biblioteka CSS. Jest on rozwijany przez programistów Twittera, wydawany na licencji MIT. Zawiera zestaw przydatnych narzędzi ułatwiających tworzenie interfejsu graficznego stron oraz aplikacji internetowych.

Instalacja (Webpack):

  1. npm install bootstrap - instalujemy w terminalu bootstrapa (w tym przypadku będzie to C:\dev\Awesome Notes)
  2. import 'bootstrap'; - dodajemy tą linijkę kodu do gównego pliku index.js
  3. npm install --save jquery popper.js - dodajemy bibliotekę jQuery oraz narzędzie do pozycjonowania Popper
  4. @import "custom"; oraz @import "~bootstrap/scss/bootstrap"; - dodajemy te linijki kodu do gównego pliku main.scss (wymagane ładowarki: sass-loader, postcss-loader z Autoprefixerem - w tym webpacku już mam)
  5. import 'bootstrap/dist/css/bootstrap.min.css'; - alternatywne rozwiązanie punktu czwartego (wymagany podstawowy style-loader oraz css-loader)
