VUE.JS

Vue.js to front-endowy framework JavaScript typu open source model-view-viewmodel do tworzenia interfejsów użytkownika i aplikacji jednostronicowych. Został stworzony przez Evana You i jest utrzymywany przez niego i pozostałych aktywnych członków zespołu.

Instalacja Vue (sposób nr.1):

  1. <script> src="https://unpkg.com/vue@next"</script> - instalacja poprzez użycie CDN package
  2. Praktyczny przykład:
    <div id ="app" > 
  <p> {{ title }} </p> 
</div>

<script>
  new Vue ({
    el: '#app',
    data: {
      title: 'Hello Vue!'
    }
  })
</script>

Instalacja Vue (sposób nr.2):

  1. npm install vue@next - instalacja poprzez użycie npm

Instalacja Vue (sposób nr.3):

  1. yarn global add @vue/cli @vue/cli-service-global lub npm install -g @vue/cli @vue/cli-service-global - instalacja poprzez użycie CLI
  2. vue create newApp - tworzymy projekt vue w nowym folderze newApp
  3. vue create . - tworzymy projekt vue w obecnym folderze
  4. Praktyczny przykład:
    <template > 
  <p> {{ title }} </p> 
</template>

<script>
  export default {
    data() {
      return {
        title: 'Hello Vue!',
      }
    }
  }
</script>

Vue - podstawy

  • {{ title }} - przykładowo dodajemy tytuł do elementu html
  • el: '' - nasza aplikacja 'app'
  • template: `kod html` - nasza kod html umieszczony w skrypcie vue
  • data: {} - tu dodajemy elementy aplikacji
  • methods: {} - tu dodajemy nasze funkcje
  • watch: {} - tu dodajemy elementy aplikacji, które chcemy obserwować
  • computed: {nazwaFunkcji(){return}} - tu dodajemy elementy, którę mają się automatycznie kalkulować

Dyrektywy

  • {{ title }} - przykładowo dodajemy tytuł do elementu html
  • v-if - instrukcja warunkowa if
  • v-else - instrukcja warunkowa else
  • v-for="name in names" - pętla for
  • v-bind:key - klucz dzięki, któremu rozpoznajemy elementy (używamy np w pętli)
  • v-on:click lub @click - przykładowo do przycisku dodajemy funkcję
  • v-model - przechwytuje zawartość inputa
  • v-bind:class="{ nazwaKlasy: coSięMaWydażyć }" - dodajemy zlinkowaną klasę (podobnie jak const.classList.toggle("nazwaKlasy"))
